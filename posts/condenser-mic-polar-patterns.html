Direction ของ Condenser Mic — ทำความเข้าใจ Polar Pattern ก่อนจะวางไมค์

โดย กองบรรณาธิการ · 13 มิ.ย. 2569 · 7 นาทีในการอ่าน

ถ้าเคยซื้อไมค์แล้วเห็นสัญลักษณ์แปลกๆ บนกล่อง หรือลองกดปุ่มบนไมค์ที่มีหลาย pattern แล้วไม่แน่ใจว่าต่างกันยังไง — บทความนี้ตอบได้ครับ Polar Pattern คือสิ่งที่บอกว่าไมค์รับเสียงจากทิศทางไหนได้บ้าง และนั่นส่งผลต่อเสียงที่ได้มากกว่าที่หลายคนคิดไว้

ไมค์ condenser ส่วนใหญ่ในท้องตลาดมี pattern ให้เลือกตั้งแต่ 1 ถึง 5 แบบ บางตัวล็อกมาเป็นแบบเดียว บางตัวสลับได้ผ่านสวิตช์ที่ตัวไมค์ การเข้าใจแต่ละแบบไม่ได้แค่ช่วยให้ใช้ไมค์ได้ถูกต้อง แต่ยังเปิดทางให้ทำเสียงที่ดีขึ้นในสถานการณ์ต่างๆ ด้วย

Cardioid — ขาใหญ่ที่ทุกคนคุ้นเคย

Cardioid

Cardioid คือ pattern ที่พบบ่อยที่สุดในไมค์สตูดิโอและ live performance รูปร่างของ polar pattern มีลักษณะคล้ายหัวใจ (cardioid มาจากภาษากรีก kardia = หัวใจ) รับเสียงเต็มที่ด้านหน้า ลดลงตามด้านข้าง และแทบไม่รับเสียงด้านหลังเลย การที่ side rejection ดีทำให้ cardioid เหมาะมากกับห้องที่ acoustics ไม่สมบูรณ์แบบ เพราะช่วยตัดเสียง room และ reflections ด้านหลังออกไปได้มาก ใช้ได้กับการร้อง, กีตาร์อะคูสติก, เครื่องดนตรีเดี่ยว และ podcast ทั่วไป

Omnidirectional — รับเสียงรอบทิศ ไม่ตัดอะไรออก

Omnidirectional

Omni รับเสียงเท่ากันทุกทิศทาง ไม่มีทิศที่ดีกว่าหรือแย่กว่ากัน ข้อดีที่หลายคนมองข้ามคือ omni ไม่มี proximity effect ซึ่งหมายความว่าเสียงที่ได้จะ natural และ flat กว่าเมื่อเทียบกับ cardioid ในระยะเดียวกัน นิยมใช้ในงาน field recording, บันทึกเสียงบรรยากาศ, conference call หรือเมื่ออยากได้ room sound เข้ามาในงาน ข้อระวังคือในห้องที่ acoustics ไม่ดี omni จะดูดเสียง reflection เข้ามาทั้งหมด ซึ่งอาจไม่ใช่สิ่งที่ต้องการเสมอไป

Figure-8 (Bidirectional) — รับสองด้าน ตัดด้านข้าง

Figure-8 / Bidirectional

Figure-8 รับเสียงด้านหน้าและด้านหลังเท่ากัน แต่ตัดเสียงด้านข้างออกอย่างสมบูรณ์ มันคือ pattern ที่ใช้ใน Blumlein stereo technique และ Mid-Side recording ที่พูดถึงในบทความก่อน ในห้อง studio ถ้ามีสองคนนั่งตรงข้ามกัน ใช้ figure-8 ตัวเดียวก็บันทึกได้ทั้งคู่ในคุณภาพเดียวกัน ข้อดีอีกอย่างของ figure-8 คือ rejection ด้านข้างแม่นมากกว่า cardioid ซึ่งมีประโยชน์มากในงานที่ต้องการ isolation สูง

Supercardioid และ Hypercardioid — เน้นหน้า ตัดข้างแน่น

Supercardioid

Supercardioid คือ cardioid ที่แคบกว่า รับเสียงด้านหน้าในมุมที่แคบลง ตัดด้านข้างได้ดีกว่า แต่มี rear lobe เล็กๆ ด้านหลัง ส่วน Hypercardioid แคบกว่า supercardioid อีก และ rear lobe ด้านหลังใหญ่ขึ้นเล็กน้อย สองแบบนี้นิยมใช้ใน live stage เพราะ off-axis rejection ดีกว่า cardioid ปกติ ทำให้ feedback loop เกิดยากขึ้น และใช้ในสถานการณ์ที่ต้องการ isolation สูง เช่น บันทึกกลองในห้องที่มีเสียงรบกวนรอบข้าง

สิ่งที่ต้องระวังเมื่อใช้ super/hypercardioid คือตำแหน่งของ monitor speaker บนเวที ต้องไม่อยู่ตรง null point ด้านข้าง แต่ต้องไม่อยู่ตรง rear lobe ด้วย ซึ่งต่างจาก cardioid ที่วาง monitor ด้านหลังไมค์ได้ตรงๆ เลย

เลือกแบบไหน — ตารางสรุปเร็ว

Pattern เหมาะกับ ระวัง
Cardioid ร้อง, podcast, กีตาร์, เครื่องดนตรีเดี่ยว Proximity effect ถ้าอยู่ใกล้เกิน
Omni Field recording, ห้องเสียงดี, conference ดูด room เข้ามาทั้งหมด
Figure-8 MS recording, Blumlein, สัมภาษณ์คู่ Side null point แม่นมาก ระวังตำแหน่ง
Supercardioid Live stage, isolation สูง มี rear lobe เล็กๆ ด้านหลัง
Hypercardioid ตัด bleed สูง, บันทึกกลอง Rear lobe ใหญ่กว่า super — ระวัง monitor

ไมค์ที่ดีไม่ใช่แค่ไมค์ที่ราคาแพงหรือยี่ห้อดัง แต่คือไมค์ที่เหมาะกับสถานการณ์และมีการวางที่ถูกต้อง ความเข้าใจ polar pattern คือพื้นฐานที่ทำให้ตัดสินใจได้ถูกต้องว่าจะวางไมค์ตรงไหน หันหน้าไมค์ไปทางไหน และจะจัดการกับเสียงที่ไม่ต้องการยังไง ก่อนซื้อไมค์ตัวต่อไป ลองถามตัวเองก่อนว่าต้องการ polar pattern แบบไหน แล้วค่อยดูสเปคอื่น
